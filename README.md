# SRI Voice Assistant

## Description
SRI is a personal voice assistant built with Python. It uses speech recognition to listen to your commands and responds with voice output. SRI can play music on YouTube, tell the time, provide information from Wikipedia, share jokes, open applications like Chrome and VS Code, and more.

## Features
- **Play Music**: Say "play [song name]" to play it on YouTube.
- **Tell Time**: Ask "what's the time" to get the current time.
- **Wikipedia Search**: Say "who is [person]" to get a summary.
- **Jokes**: Say "joke" to hear a random joke.
- **Open Apps**: Say "open chrome" or "open code" to launch applications.
- **Exit**: Say "exit" or "stop" to quit the assistant.

## Requirements
- Python 3.x
- Libraries: speech_recognition, pyttsx3, pywhatkit, wikipedia, pyjokes
- Microphone for voice input
- Internet connection for some features (e.g., YouTube, Wikipedia)

## Installation
1. Clone or download the repository.
2. Install dependencies: `pip install speech_recognition pyttsx3 pywhatkit wikipedia pyjokes`
3. Run the script: `python sri_assistant.py`

## Usage
1. Run the script.
2. SRI will greet you and start listening.
3. Speak commands clearly into your microphone.
4. Enjoy interacting with SRI!
